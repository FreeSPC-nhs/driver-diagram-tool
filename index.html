      Help & information

      Using this tool

      This web-based driver diagram tool lets you create and edit a structured driver diagram made up of an Aim, Primary drivers, Secondary drivers, and Change ideas. The diagram updates instantly as you edit, reorder, colour, or resize items.

      1. Understanding the diagram structure

      • Aim: the overall goal you are trying to achieve (left-hand column).
      • Primary drivers: the main factors that directly contribute to the aim.
      • Secondary drivers: factors that influence the primary drivers.
      • Change ideas: specific actions or tests of change that influence drivers.

      The diagram shows cause and effect from right to left: if we implement these change ideas, they influence our secondary and primary drivers, which together help us achieve the aim.

      2. Adding items

      • Use the + buttons on any box to add a child item at the next level.
      • New items automatically appear in the correct column and connect to their parent.
      • You can also add items from the controls panel if preferred.

      3. Editing items

      • Click directly on an item’s text to edit it.
      • Right-click (or use the context menu) on any item for more options.
      • You can edit column headings by clicking on the heading text.

      4. Reordering items

      • Right-click an item and use Move up, Move down, Move to top, or Move to bottom.
      • Items are reordered within their level and parent group.
      • The diagram automatically redraws to maintain correct branch alignment.

      5. Auto-sizing boxes

      • If text is long, right-click the item and choose Toggle auto-size to text.
      • The box will expand vertically to fit its content.
      • The diagram layout automatically adjusts so connections remain aligned.

      6. Using colours

      • Add, rename, or delete colours in the colour management panel.
      • Right-click any box and use Change colour ▸ to select from your defined colours.
      • Colours can help group themes, workstreams, priorities, or ownership.
      • The legend updates automatically to reflect your palette.

      7. Connections and structure

      • Items are connected automatically to their parent.
      • You can add or remove additional connections using the connector button.
      • The diagram maintains parent–child structure while allowing flexible linking.

      8. Deleting items

      • Right-click an item and choose Delete….
      • Deleting an item also removes its dependent child items.
      • Connections associated with deleted items are removed automatically.

      9. Appearance settings

      • Adjust box height, vertical spacing, font size, font family, and bold styling.
      • Appearance settings apply globally to the diagram.
      • Auto-sized boxes will expand beyond the base height if needed.

      10. Exporting and saving

      • Export as PNG for presentations or reports.
      • Export as PDF for sharing or printing.
      • Download and upload CSV files to save your work.
      • CSV export preserves structure, colours, appearance settings, and layout order.

      This tool is designed to support continuous learning. As you test change ideas and review data (for example using run charts or SPC), you can update the diagram to reflect your latest understanding.
